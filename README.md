# チームシャッフル（Vite + React）

参加者を4人パーティーに分け、実力バランス・ペア重複回避・リーダー偏りを抑えるローカルWebアプリです。OCR（tesseract.js）で画像から参加者候補も抽出できます。

## 起動方法

```bash
npm install
npm run dev
```

## 使い方

1. **参加者入力**
   - 1行につき `name, rank` を入力します。
   - rankは `S/A/B/C/D` のみです。
   - 人数は 8/12/16/20 名に対応しています。
2. **ラウンド数・生成回数** を調整します。
3. **生成** を押してラウンドを作成します。気に入らなければ **もう一回生成** で再実行できます。
4. **Roundごとのコピー** ボタンでDiscord向けのテキストをコピーできます。
5. **OCR**
   - 画像をドラッグ&ドロップ、または貼り付けて解析します。
   - 候補を編集し、**候補を参加者欄に反映** で `name, B` 形式の入力に変換できます。

## 注意点

- OCR精度は画像の解像度や背景に左右されます。結果は必ず目視で確認してください。
- ロジックはローカルで完結し、外部サービスへの送信はありません。
